En esta aplicación web, presento una tienda virtual completa.

Esta aplicación web depende de una base de datos que usa mySQL database. Para ello he usado phpmyadmin (XAMPP).

-En la carpeta "base de datos" se encuentra el código SQL con las tablas guardadas.

-En el .txt llamado usuarios y contraseñas, se pueden ver los usuarios y contraseñas guardadas en la base de datos.
